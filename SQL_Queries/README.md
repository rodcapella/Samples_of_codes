# SQL Queries Collection

This directory contains a collection of **SQL query examples and scripts** used for learning, experimentation, and reference.
The queries demonstrate common database operations and patterns frequently used in **data analysis, reporting, and backend systems**.

## Purpose

The goal of this folder is to provide **practical SQL examples** that can be reused or adapted for different relational database systems such as:

* MySQL
* PostgreSQL
* SQL Server
* SQLite
* Oracle

These scripts may include examples of:

* Data retrieval
* Filtering and sorting
* Aggregations
* Joins
* Subqueries
* Data manipulation
* Query optimization patterns

## Structure

Each file in this folder contains **independent SQL queries** that demonstrate specific techniques or solve particular database problems.

## Topics Covered

Typical SQL concepts that may appear in this directory include:

* `SELECT` statements
* `WHERE` filtering
* `GROUP BY` and aggregations (`COUNT`, `SUM`, `AVG`)
* `ORDER BY`
* `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`
* Subqueries
* Conditional expressions (`CASE`)
* Data transformations
* Query optimization patterns
* Windows Functions (CTEs)

## How to Use

1. Open any `.sql` file using your preferred SQL editor or database client.
2. Connect to your database.
3. Execute the queries individually or as a script.

## Requirements

You need access to a relational database system such as:

* MySQL
* PostgreSQL
* SQL Server
* SQLite

and a SQL client such as:

* DBeaver
* DataGrip
* MySQL Workbench
* psql
* SQL Server Management Studio
